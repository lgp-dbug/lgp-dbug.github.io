fabPreview

The requested solution consists in providing the user the option to fast forward a simulation of a scenario generated by cmNavigo's Scheduler. The simulation is presented in the fabLive module, and the user can specify the simulation speed. This allows an easy perception of bottlenecks on materials or other problems in a given scenario.

CRITICAL Manufacturing

CRITICAL Manufacturing provides innovative software for High-Tech industries, providing a manufacturing execution and intelligence system that empowers organizations to achieve its goals, designed cmNavigo.

Recent versions of this system contain a Scheduler module, which generates a scenario and associates materials to the respective resources, maximizing production parameters specified by the user. fabLive is another module of cmNavigo, which is capable of presenting the user real-time changes on the factory.

Statium

O Statium é um portal de estatísticas de futebol proposto pela Bitmaker Software, que será criado com recurso a um dataset de grandes dimensões sobre factos de jogos de uma liga desportiva. Através de ferramentas de Business Intelligence, o portal irá permitir aos utilizadores a visualização dos dados com gráficos interativos e multidimensionais, facilitando a introspeção de uma quantidade vasta de informação e também a descoberta de conhecimentos previamente ocultos nos dados.

[Altran]

Este projeto foi proposto pela Altran e tem como objetivo colmatar uma falha tecnológica que os mesmos possuem. É uma empresa de Consultoria de Inovação e Tecnológica. Está envolvida em bastantes projetos e para melhor servirem os seus clientes, dispõem de inquéritos de satisfação em papel. Isto torna-se pouco prático para cruzamento de dados. A solução é oferecer uma plataforma web ao cliente, onde este poderá visualizar os seus projetos, que estejam na recta final, e responder diretamente ao inquérito de satisfação.

  • Client: Start Bootstrap
  • Date: April 2014
  • Service: Web Development

[Life Iberia]

O projecto requisitado pela empresa msg Life Iberia pode ser dividido em dois subprojectos, com requisitos diferentes mas um objectivo mútuo: divulgar os produtos da empresa em redes sociais. O primeiro consiste numa ferramenta web que permita a difusão de anúncios em redes sociais como o Facebook e o Google+, permitindo ao utilizador escolher que secções da população considera o alvo. O segundo consiste numa aplicação Canvas de Facebook cujo propósito é permitir a qualquer utilizador com uma conta Facebook simular um produto da msg Life Iberia através do preenchimento de um formulário.

  • Client: Start Bootstrap
  • Date: April 2014
  • Service: Web Development
